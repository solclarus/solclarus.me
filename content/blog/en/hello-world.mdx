## Introduction

Hi, I'm solclarus. In this blog, I'll be writing about software development and technology.

I mainly work on web development, covering everything from frontend to backend. I plan to use this blog as a place to try out new technologies and document insights gained during development.

## About This Blog

Here's what you can expect from this blog:

### Technical Learnings

I'll share things I learn during daily development and my impressions of trying new libraries and frameworks. I'm particularly interested in:

- **Frontend**: React, Next.js, TypeScript
- **Styling**: Tailwind CSS, CSS-in-JS
- **Tools**: Bun, Vite, Docker

### Project Documentation

I'll document the development process of personal projects, reasons for technology choices, and challenges I faced. I hope this can be helpful to others facing similar issues.

### Daily Insights

Beyond development, I'd also like to write about workflows for improving productivity and introduce useful tools.
