## はじめに

こんにちは、solclarus です。このブログでは、ソフトウェア開発や技術について書いていきます。

普段は Web 開発を中心に、フロントエンドからバックエンドまで幅広く触れています。新しい技術を試したり、開発中に得た知見を記録したりする場所として、このブログを活用していく予定です。

## このブログについて

このブログでは、以下のような内容を書いていきます。

### 技術的な学び

日々の開発で学んだことや、新しいライブラリ・フレームワークを試した感想などを共有します。特に以下の分野に興味があります：

- **フロントエンド**: React, Next.js, TypeScript
- **スタイリング**: Tailwind CSS, CSS-in-JS
- **ツール**: Bun, Vite, Docker

### プロジェクトの記録

個人プロジェクトの開発過程や、技術選定の理由、つまずいたポイントなどを記録していきます。同じような課題に直面している方の参考になれば幸いです。

### 日々の気づき

開発以外にも、生産性を上げるためのワークフローや、便利なツールの紹介なども書いていきたいと思っています。

## 技術スタック

このブログ自体も、学びの一環として最新の技術スタックで構築しています：

```
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- MDX

```

詳しい技術スタックについては、[別の記事](/blog/tech-stack)で紹介しています。
