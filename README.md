# solclarus.me

🌐 https://solclarus.me

## Stack

- **Next.js 16** (App Router)
- **Tailwind CSS v4**
- **next-intl** (ja / en)
- **MDX** blog
- Bun / oxlint / oxfmt

## Development

```bash
bun install
bun run dev
```

### Environment Variables

```bash
# .env.local
GITHUB_TOKEN=   # read:user — for contribution heatmap
```

## Commands

```bash
bun run dev        # dev server
bun run build      # production build
bun run lint       # oxlint
bun run format     # oxfmt
```
